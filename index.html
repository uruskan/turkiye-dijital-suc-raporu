Altyapı Bulguları
  • Cloudflare Dominansı: Yasadışı bahis sitelerinin %76+'sı Cloudflare arkasında (Endonezya analizi referans). IP maskeleme ve DDoS koruması için tercih ediliyor.

  • Papara Vakası: 55.09 Milyar TL şüpheli işlem hacmi, 247.886 hesap.

2.3 DNS TTL Patterns ve Tespit Fırsatları

Domain yaşam döngüsü boyunca TTL (Time-To-Live) değerleri değişir. Bu pattern'ler tespit için önemli sinyaller sağlar:

Aşama TTL Değeri Süre Karakteristik Tespit Fırsatı
Registration N/A T+0 Domain yeni kaydedildi WHOIS monitoring, Certificate Transparency logs, new domain feeds
Staging 300-600s (5-10 dk) 0-24 saat Test edilıyor, içerik yükleniyor Passive DNS, SSL cert issuance alerts, low traffic anomaly
Active 60-300s (1-5 dk) 1-14 gün Aktif kullanımda, trafik alıyor Content fingerprinting, user reports, traffic analysis
Evasion 30-60s (30-60 sn) Sürekli Fast-flux, IP rotasyonu Fast-flux detection algorithms, DNS query volume spike
Burned N/A Post-block Engellenmiş, terk edilmiş Mirror domain'e geçiş izleme, CNAME chain takibi
Tespit Kuralı: TTL < 120 saniye + domain age < 30 gün + gambling keyword pattern = Yüksek güvenilirlikli IOC

2.6 Domain IOC Patterns

# HIGH CONFIDENCE PATTERNS

# Pattern 1: Brand + Sequential Number
Regex: (betist|bets10|matbet|bahiswin|superbet|casino)\d{2,4}\.(com|net|xyz|top|win|bet)
Examples: betist789.com, matbet456.net, bets10123.xyz
Confidence: HIGH

# Pattern 2: Giris/Login/Guncel Prefix
Regex: (giris|login|guncel|yeni|mobil|m)[-_]?(betist|matbet|bahis|casino)\.(com|xyz|info|top)
Examples: giris-betist.xyz, yeni-matbet.top, m-casino.info
Confidence: HIGH

# Pattern 3: Mirror/Yedek Variants
Regex: (mirror|ayna|yedek|alternatif)\d?[-.]?\w+\.(com|xyz|top)
Examples: mirror1-betist.xyz, ayna-matbet.com, yedek-bets10.top
Confidence: MEDIUM

# Pattern 4: Typosquatting
Regex: (bet1st|bettist|betis|betsiten|mathbet|matbett|casin0)\.(com|net)
Examples: bet1st.com, mathbet.net, casin0.xyz
Confidence: MEDIUM

# Pattern 5: Turkish Keyword Stuffing
Regex: (bahis|kumar|casino|slot)[-_]?(bonus|kazan|para|bedava)[-_]?(tr|turk)?\.(com|xyz|top)
Examples: bahis-bonus-tr.xyz, casino-kazan.com, slot-bedava.top
Confidence: MEDIUM

# Pattern 6: Free Hosting Abuse
Regex: \w+\.(blogspot|weebly|wixsite|000webhostapp|netlify\.app|vercel\.app)
Examples: bahisfirsat.blogspot.com, casino-bonus.netlify.app
Confidence: LOW (requires content analysis for confirmation)

3. Şantaj/Sextortion Operasyonları

Bu bölüm, Türkiye'deki sextortion (cinsel şantaj) operasyonlarının metodolojisini, kullanılan OSINT tekniklerini, saldırı zincirini ve savunma yöntemlerini analiz etmektedir.

Ölçek: Türkiye'de yıllık tahmini 5.000+ bilinen sextortion vakası bulunmaktadır. Gerçek sayı, utanç ve korku nedeniyle bildirmeyen mağdurlar dahil edildiğinde çok daha yüksektir. Ortalama talep: 50.000 - 500.000 TL.

# RULE: RAPID_FUND_MOVEMENT

    meta:
        description = "Detects rapid in-out pattern"
        severity = "CRITICAL"
    
    condition:
        FOR ANY deposit IN deposits(window=24h):
            EXISTS withdrawal IN withdrawals(window=30min, after=deposit):
                withdrawal.amount >= deposit.amount * 0.90 AND
                withdrawal.destination NOT IN account.known_recipients
    
    action:
        hold_withdrawal()
        request_source_verification()
        alert(team="AML", priority="HIGH")
}

# RULE: DEVICE_FRAUD_RING
rule DEVICE_CLUSTER {
    meta:
        description = "Detects multiple accounts on single device"
        severity = "CRITICAL"

            SELECT fingerprint FROM devices 
            GROUP BY fingerprint
